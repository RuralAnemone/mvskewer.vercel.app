## to deploy:
- optional (for now?): 
  - copy `.env.example` to `.env.local`
  - fill in with your openai api key
- run `yarn install`
- run `yarn build`
- run `yarn start`

### or, run this command:
```bash
cp .env.example .env.local \
vim .env.local \
yarn \
yarn build \
yarn start
```
